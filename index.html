    Klasifikasi dan Analisis Kemiripan Gen Kloroplas Famili Sapindaceae Menggunakan Pendekatan Machine Learning (Classification and Similarity Analysis of Chloroplast Genes in the Sapindaceae Family Using a Machine Learning Approach)

  • Final project with my group consist of 4 peoples for Computational Biology Subject in my 4th Semester.
  • We extracted several features from the chloroplast DNA of plants in the Sapindaceae family and those outside the family. These features were used to classify plants based on their chloroplast DNA. We performed the classification using a machine learning approach, specifically employing the random forest, histogram gradient boost, and AdaBoost algorithms. Our results showed that the random forest algorithm performed the best according to the classification report displayed.
  • We are using Python.
